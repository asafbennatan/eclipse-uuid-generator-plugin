    • Eclipse UUID generator plugin

    Generates UUIDs and pastes them to the editor

    Project maintained by madhead

    Installation

    Using ZIP distribution

    1. Download the latest zip archive with the plugin.
    2. You'll find a single JAR file inside plugins directory in it.
    3. Place this JAR to the dropins directory of your Eclipse installation.
    4. (Re)?start Eclipse.

    Using the update site

    1. Go to Help -> Install New Software.
    2. Work with http://madhead.github.io/eclipse-uuid-generator-plugin/updatesite/.
    3. Select UUID generator.

    Usage

    Just place the caret in editor to any place you want to insert the UUID and press Ctrl + Alt + U. You can change the shortcut in your settings, the command name is Generate UUID.
